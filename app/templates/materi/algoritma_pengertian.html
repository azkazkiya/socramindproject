Pengertian Algoritma

Algoritma adalah serangkaian instruksi atau langkah-langkah logis yang disusun secara sistematis untuk menyelesaikan suatu masalah atau mencapai tujuan tertentu. Algoritma dapat ditulis dalam bentuk koding maupun pseudo-code yang tujuan akhirnya agar mudah dipahami dan diimplementasikan oleh komputer. Dalam kehidupan sehari-hari, pemrograman sering digunakan dalam berbagai bidang.

Bayangkan seperti resep masakan, di mana setiap langkah harus diikuti secara berurutan untuk menghasilkan hidangan yang lezat. Algoritma dirancang untuk memecahkan masalah yang sederhana maupun masalah yang bersifat kompleks. Algoritma menjadi aspek paling penting saat developer membuat program dengan menggunakan bahasa pemrograman apapun. Pasalnya, hasil akhir dari program yang dibuat tersebut tergantung dari algoritma yang telah ditulis.

Fungsi Algoritma

Fungsi utama dari algoritma sendiri adalah membantu para programmer dalam merancang maupun menulis kode secara efektif, efisien dan bisa dipahami dengan mudah. Selain yang sudah disebutkan di atas, berikut adalah beberapa fungsi lain dari algoritma pemrograman yang wajib kamu ketahui.

  1. Menyederhanakan program yang rumit dan besar menjadi lebih sederhana, sehingga penggunaannya bisa lebih efektif dan efisien.
  2. Memudahkan untuk cepat menemukan kesalahan dalam program saat terjadi eror dan bisa segera diatasi segera karena alurnya yang jelas dan runtut.
  3. Proses modifikasi menjadi lebih mudah karena tidak perlu merubah seluruh modul, namun cukup mengubah modul yang berkaitan saja.
  4. Membantu menyelesaikan permasalahan yang kompleks pada programs serta meminimalisir kesalahan penghitungan matematis tingkat tinggi.
  5. Script yang telah dibuat secara urut dan sistematis membuat proses pengembangan menjadi lebih mudah.
  6. Bisa digunakan berulang kali dan tidak perlu repot menuliskan kembali program yang sama, sehingga lebih memudahkan dalam membuat program.

Tipe Data dalam Variabel dan Operator

a. Tipe Data

Dalam membuat kode program, sangat perlu untuk memperhatikan tipe data yang sesuai dengan nilai yang akan digunakan dalam sebuah variabel. Variabel akan digunakan dalam setiap operasi, dan jika tipe datanya tidak sesuai, program bisa mengalami eror.

Kita bisa membayangkan tipe data sebagai "wadah". Ada wadah khusus untuk air, pasir, dan surat. Kita tidak bisa menyimpan air dalam wadah surat, kan? Begitu pula dalam pemrograman, setiap data memiliki jenisnya sendiri.

Berikut adalah empat tipe data paling dasar yang akan sering kita gunakan:

Integer (int)

Tipe data ini digunakan untuk menyimpan bilangan bulat (positif, negatif, atau nol). Contoh: 17, -45, 0.


# Menyimpan umur dalam tipe data integer
umur = 17
print(f"Umur saya adalah {umur} tahun")
Float

Tipe data ini digunakan untuk menyimpan bilangan desimal atau pecahan. Contoh: 3.14, 99.5, -0.25.


# Menyimpan tinggi badan dalam tipe data float
tinggi_badan = 172.5
print(f"Tinggi badan saya {tinggi_badan} cm")
String (str)

Tipe data ini digunakan untuk menyimpan teks atau kumpulan karakter. Nilainya selalu diapit oleh tanda kutip (" atau '). Contoh: "Halo Dunia", 'SMK Bisa!'.


# Menyimpan nama dalam tipe data string
nama = "Budi Sanjaya"
print(f"Nama lengkap saya adalah {nama}")
Boolean (bool)

Tipe data ini sangat spesial karena hanya bisa memiliki dua nilai: True (Benar) atau False (Salah). Tipe data ini adalah jantung dari logika percabangan.


# Menyimpan status kelulusan dalam tipe data boolean
apakah_lulus = True
print(f"Status kelulusan: {apakah_lulus}")
b. Operator

Operator adalah simbol-simbol khusus yang digunakan untuk melakukan operasi, seperti perhitungan matematika atau perbandingan logika.

Operator Aritmatika

Operator ini digunakan untuk melakukan operasi matematika dasar.

Operator Nama Contoh Hasil
+ Penjumlahan 10 + 5 15
- Pengurangan 10 - 5 5
* Perkalian 10 * 5 50
/ Pembagian 10 / 5 2.0
% Modulus (Sisa Bagi) 10 % 3 1
Operator Perbandingan

Operator ini digunakan untuk membandingkan dua nilai. Hasil dari operasi ini selalu bernilai Boolean (True atau False).

Operator Arti Contoh Hasil
==Sama dengan5 == 5True
!=Tidak sama dengan5 != 3True
>Lebih besar dari5 > 3True
<Lebih kecil dari5 < 3False
>=Lebih besar atau sama dengan5 >= 5True
<=Lebih kecil atau sama dengan5 <= 3False

Penting: Jangan tertukar antara = (untuk memberi nilai ke variabel) dan == (untuk membandingkan)!

Operator Logika

Operator ini digunakan untuk menggabungkan beberapa kondisi Boolean.

Operator Cara Kerja Contoh Hasil
and Akan menghasilkan True hanya jika kedua kondisi bernilai True. (5 > 3) and (10 == 10) True
or Akan menghasilkan True jika salah satu kondisi bernilai True. (5 < 3) or (10 == 10) True
not Akan membalikkan nilai Boolean. not (5 < 3) True

Contoh Kasus: Menghitung Luas Persegi Panjang

Misalkan kita ingin membuat program untuk menghitung luas persegi panjang. Langkah pertama adalah mengetahui proses logisnya terlebih dahulu, mulai dari rumus hingga cara menampilkannya. Jika sudah, kita bisa mengubahnya ke dalam bentuk algoritma dan kode.

Algoritma Mencari Luas Persegi Panjang
  1. Mulai
  2. Siapkan 'wadah' (Tipe data dan variabel) bernama panjang, lebar, dan luas.
  3. Minta pengguna memasukkan nilai untuk panjang.
  4. Minta pengguna memasukkan nilai untuk lebar.
  5. Hitung luas dengan mengalikan panjang dan lebar.
  6. Tampilkan hasil luas.
  7. Selesai
